## 📖 Descripción del Proyecto

Analizar ventas de piezas de motocicletas a lo largo del tiempo y en múltiples almacenes. El objetivo es identificar tendencias, comparar el rendimiento entre warehouses y derivar insights accionables para el negocio.

---

## 🗃️ Tabla: `sales`

| Columna | Descripción |
|---------|-------------|
| `order_number` | Referencia única del pedido |
| `date` | Fecha del pedido |
| `warehouse` | Almacén de origen (North / Central / West) |
| `client_type` | Tipo de cliente (Retail / Wholesale) |
| `product_line` | Línea de producto |
| `quantity` | Cantidad vendida |
| `unit_price` | Precio unitario |
| `total` | Total de la venta |
| `payment` | Método de pago |
| `payment_fee` | Porcentaje de comisión por método de pago |
