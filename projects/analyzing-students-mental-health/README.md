## 📖 Descripción del Proyecto

Aplicar habilidades de manipulación de datos para realizar un análisis exploratorio sobre la salud mental de estudiantes internacionales.

El dataset proviene de un estudio de una universidad japonesa (2018) que evaluó si estudiar en el exterior afecta la salud mental. Se analizan factores como el diagnóstico de depresión, estrés de adaptación social e idioma, y dificultades de conexión.

---

## 🗃️ Tabla: `students`

| Columna | Descripción |
|---------|-------------|
| `inter_dom` | Tipo de estudiante: internacional o doméstico |
| `japanese_cate` | Nivel de japonés del estudiante |
| `english_cate` | Nivel de inglés del estudiante |
| `academic` | Nivel académico actual |
| `age` | Edad del estudiante |
| `stay` | Años de permanencia en la universidad |
| `todep` | Puntuación de depresión (PHQ-9) |
| `tosc` | Puntuación de conexión social (SCS) |
| `toas` | Puntuación de estrés de adaptación (ASISS) |
